Introduction

Repérage permet de créer des exercices interactifs de repérage dans un texte : le but est de sélectionner les termes qui respectent une consigne donnée.

Pour voir un exemple d'exercice de repérage : menu Actions / Exemple de reperage ;

Le présent générateur, qui permet de créer les données textuelles de chaque type d'exercice, est une application Web en ligne ou intégrée à Chrome.

Pour créer un exercice de repérage : menu Actions / Nouvel exercice…

À l'issue de cette saisie, les données du nouvel exercice seront automatiquement enregistrées dans l'espace privé de l'application, ainsi que disponibles sous forme d'un bloc de texte à récupérer dans un éditeur de texte.

Pour modifier un exercice précédent   : menu Actions / Editer…

Principe

L'exercice s'exécute au moyen d'un fichier de données data.json et d'un exerciseur : le fichier d'interface reperage.html.

Pour créer le fichier de données data.json, vous devez utiliser le présent Générateur.

Vous téléchargerez ensuite une copie de l'exerciseur approprié, reperage.zip.

Pour publier votre exercice, vous décompresserez d'abord reperage.zip, vous copierez le fichier data.json dans le dossier décompressé reperage.

Le dossier de l'exercice, ainsi complété, sera finalement publié sur un serveur de cours et vous ferez finalement un lien vers le fichier reperage.html.

N.B.: Pour déplacer facilement le dossier complété, utilisez un logiciel d'archivage comme 7zip. Vous penserez enfin à décompresser cette archive une fois qu'elle sera déposée sur le serveur.

Pas à pas

1. Créez et récupérez le texte du fichier de données data.json grâce au Générateur.

2. Récupérez une copie de l'exerciseur Repérage reperage.zip et décompressez-la.

3. Copiez votre fichier data.json dans le dossier reperage.

4. Renommez le dossier reperage sous un nom plus explicite.

5. Créez une nouvelle archive .zip du dossier de reperage renommé.

6. Déposez cette archive votre_reperage.zip à l'endroit désiré sur le serveur de cours.

7. Décompressez l'archive votre_reperage.zip sur ce serveur.
Voyez à ce propos la documentation de votre plate-forme de cours ENT.

8. Faites enfin un lien vers le fichier reperage.html.
Voyez encore la documentation de votre plate-forme de cours ENT.

Découper le texte

Segmentez votre texte en insérant une étoile (*) entre chaque groupe ou mot à isoler.
Vous pouvez aussi forcer un retour à la ligne dans l'exercice final en insérant le caractère dièse (#) devant chaque début de paragraphe envisagé.

Par exemple :
Terme un * terme deux * #Un nouveau paragraphe * terme quatre

Choisir des termes

Une fois que le texte a été découpé, les termes se présentent sous forme de liste. Il suffit alors de cocher les options en face des termes qui respectent la consigne.

Enrichir le commentaire de fin d'exercice

Pour mettre en évidence une expression dans le commentaire final, placez deux caractères de soulignement (__) au début de l'expression et un seul caractère de soulignement pour achever cette mise en évidence.
Par exemple : Ceci apparaîtra __en évidence_ dans le commentaire.

Importer un exercice enregistré

Ouvrez un fichier d'exercice (.json) dans un éditeur ou le bloc-notes.

Sélectionnez le contenu du fichier et copiez-le dans le presse-papier.

Dans le générateur : menu Actions / Importer …

Collez le code dans la zone de texte. Cliquez sur le bouton Importer.

Récupérer le code généré

Copiez ce code dans le presse-papier : édition / copier.

Ouvrez un fichier vierge dans un éditeur de texte : Bloc-Notes, TextEdit, etc.

Collez le contenu du presse-papier dans ce fichier : édition / coller.

Enregistrez ce fichier sous le nom : data.json.

Revenez dans le générateur de reperage.

Procurez-vous une copie de l'exerciseur, reperage : menu Actions / Copie de cet exerciseur….

Télécharger un exerciseur

Obtenez une copie de l'exerciseur reperage reperage.zip
en cliquant sur le bouton prévu   : menu Actions / Exerciseur …

Décompressez cette archive. Vous obtenez un dossier   : reperage   ;

Déplacez-y le fichier data.json.

Pour publier cet exercice de Repérage sur un serveur, renommez, puis compressez à nouveau ce dossier reperage mis à jour.
